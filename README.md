# GDG Week 5 Notes

## Goals

1. Create a GitHub repository with README.md file
2. Open the repository in a GitHub Codespace
3. In Copilot, enter "hello" to make sure that everything is working
4. Create a website by using the following prompt in Copilot:
   ```text
   Using React, create a website that performs <insert your description here>, then deploy it to Vercel.
   ```
5. Push your GitHub repo by doing the following:
   ```zsh
   git add .
   git commit -am "Initial commit."
   git push
   ```
7. On Vercel.com, sign up or sign in using GitHub.
8. Click **Add New...**
9. Click **Project**
10. Click **Import**
11. Deploy
12. Connect personal domain to Vercel project. (optional)

## Have Questions

Just ask Copilot or another LLM for the answer.  Good luck and Happy Vibing!!!! 👍🏾
